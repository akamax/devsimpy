Threading Backend¶

class threadingBackend.ThreadingBackend(subsystem, args)[source]¶

Wrapper around the actual threading backend. It will also handle interrupts and the passing of them to the calling thread.

__init__(subsystem, args)[source]¶

Constructor.

Parameters:
  • subsystem – string specifying the subsystem to use: python, tkinter or loop
  • args – all additional arguments that should be passed to the subsystem’s constructor (must be a list)
interrupt(value)[source]¶

Interrupt a running wait call.

Parameters:value – the value that interrupts
step()[source]¶

Perform a single step in the simulation.

Warning

Only to be used with the Game Loop subsystem!

wait(time)[source]¶

Wait for the specified time using the specified subsystem; interrupts are possible to stop sooner.

Parameters:time – time to wait in seconds, a float is possible
Returns:None if the wait was not interrupted; the value passed in the interrupt if the wait was interrupted.
